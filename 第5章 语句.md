# 第5章 语句

### 5.3.1 var

全局变量是全局对象的属性。然而和其他全局对象属性不同的是，var声明的变量是无法通过delete删除的。

    (function() {
        for(var i = 0; i < 10; i++) {
            console.log(i);
        }
        console.log("========================");

        for(var i = 0, j = 10; i < 10; i++,j--) {
            console.log(i * j);
        }
        console.log("========================");

        var o = {
            x: 1,
            y: 2,
            z: 3
        };

        for(var i in o) {
            console.log(i);
        }
        console.log("========================");

        var arr = [1,2,3,4,5];

        for(var i in arr) {
            console.log(i);
        }
    }());

<b>注意：多次声明同一个变量是无所谓的。</b>
